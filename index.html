Crop & soil guide

Two parts: a vanilla crop guide that applies with no mods at all, then a Precision Farming section below that only applies if that DLC/mod is installed and enabled. They're kept separate on purpose so it's always clear which parts need the mod.

Vanilla crop guide (base game, no mods needed)

Planting/harvest windows, required equipment, and approximate sale price per 1000L for every crop in base FS25. Growth windows assume the default (non-seasonal) calendar.

Figures gathered from community FS25 guides as a reference starting point — treat exact prices/yields as approximate, they can vary with game version and economy difficulty.

⚠ Mod required, not base game
Everything below this point (soil texture, pH targets, nitrogen targets) comes from the Precision Farming DLC/mod for Farming Simulator 25. None of this applies if you're playing without that mod installed — it is not part of base FS25.
Soil types

Target pH and base yield multiplier per soil texture (Precision Farming).

Nitrogen fertilization guide

N Target (ideal nitrogen level) and N Reduction (the level below which yield starts dropping) per crop, per soil texture. Crops marked OVERFERT can benefit from being fertilized past the target; HARVESTFERT crops can also be fertilized again after harvest.

Fertilizer headroom & over-fertilization

"Net left" is roughly how much more nitrogen (in spreader units) a field can still take after reaching N Target before it's wasted. "Over-fert effect?" shows whether pushing past target actually changes anything for that crop. Oilseed Radish is a Precision-Farming-only cover crop — it doesn't exist as a sellable crop in vanilla, it's grown between seasons to fix nitrogen and earn an eco subsidy.

This guide is a reference starting point and may be refined further — if any numbers look off for a crop/soil combo, let Keiran know and it'll get corrected here for everyone.
